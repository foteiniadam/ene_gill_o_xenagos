Απαιτούμενες βιβλιοθήκες για το Arduino
Για τη μεταγλώττιση και εκτέλεση του κώδικα Arduino απαιτούνται οι παρακάτω βιβλιοθήκες:
• Wire
Βασική βιβλιοθήκη επικοινωνίας I2C του Arduino. Περιλαμβάνεται στο Arduino IDE και δεν απαιτείται ξεχωριστή εγκατάσταση.
• Adafruit PWM Servo Driver Library
Χρησιμοποιείται για τον έλεγχο του PWM driver (PCA9685) που οδηγεί τους κινητήρες του οχήματος μέσω I2C.
Η βιβλιοθήκη μπορεί να εγκατασταθεί μέσω του Library Manager του Arduino IDE.
